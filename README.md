## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/animated-accordion.git
```

Go to the project directory

```bash
  cd animated-accordion
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run android
```
